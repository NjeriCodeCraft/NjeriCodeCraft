### 🚀 Featured Projects

#### ♻️ [WasteLink — Image Validation API](https://github.com/NjeriCodeCraft/WasteLink-Image-Validation-API) | *Final Year Project*
* **The Problem:** Manual waste sorting is inefficient and slows down recycling systems.
* **The Solution:** A FastAPI-based machine learning service that classifies waste images into five categories (General, Metal, Organic, Paper, Plastic) in real time, built to power a mobile waste management app.
* **Tech:** Python, FastAPI, TensorFlow, MobileNetV2, Render (Cloud Deployment).

#### 🦠 [COVID-19 Global Data Tracker](https://github.com/NjeriCodeCraft/COVID-19-Global-Data-Tracker-plp-python-final-project) | *PLP Python Final Project*
* **The Problem:** Understanding global health trends requires clean, visual, and accessible data analysis.
* **The Solution:** A data analysis project tracking global COVID-19 trends — cases, deaths, recoveries, and vaccinations across countries and time periods. Includes data cleaning, exploratory analysis, and visualizations.
* **Tech:** Python, Pandas, NumPy, Matplotlib, Seaborn.

#### 🌾 [AgriScore-Pay](https://github.com/NjeriCodeCraft/agriscore-pay) | *Hackathon Project*
* **The Solution:** An AI-powered agricultural credit scoring and input marketplace. Leverages Interswitch APIs and predictive yield modeling to provide data-driven financing and automated harvest-revenue splits for smallholder farmers.
* **Tech:** JavaScript, Node.js, AI/ML Integration, Interswitch APIs.

#### 🏘️ [Rent Radar](https://github.com/NjeriCodeCraft/rent-radar) | *PLP Capstone Project*
* **The Solution:** A comprehensive full-stack rental platform helping budget-conscious tenants find homes with smart filtering, location-based search, and agent services.
* **Tech:** JavaScript, Node.js, MongoDB, REST APIs.

#### 📅 [FundiFix — Booking Bot](https://github.com/NjeriCodeCraft/Booking_Bot) | *Hackathon — 3rd Place 🏆*
* **The Solution:** An AI-powered scheduling assistant for local service providers (fundis & freelancers) to manage client appointments through a conversational interface.
* **Tech:** Telegram Bot API, Python, Automation Logic.

#### 🤖 [AI Resume Matcher](https://github.com/NjeriCodeCraft/ai-resume-matcher) | *University Final Year Project*
* **The Problem:** Students struggle to align their CVs to job descriptions in a competitive market.
* **The Solution:** A web application using NLP to analyze resumes against job descriptions, identify skill gaps, and auto-generate optimized resumes.
* **Tech:** Python, Flask, NLP, scikit-learn.
